                        🚀 QUICK START (3 STEPS)
================================================================================

STEP 1: Setup Gmail (One-time only)
───────────────────────────────────

1. Go to: https://myaccount.google.com/security
2. Enable "2-Step Verification"
3. Go to: https://myaccount.google.com/apppasswords
4. Generate App Password (16 characters)
5. Copy the password

⚠️ IMPORTANT: Use App Password, NOT regular password!

STEP 2: Run Email Sender
────────────────────────

Choose ONE of these commands:

Option A (Recommended):
$ python quick_email_sender.py

Option B (Direct):
$ python email_sender.py

Option C (Parallel):
$ python run_scraper_and_sender.py

STEP 3: Enter Configuration
────────────────────────────

When prompted, enter:
📧 Gmail address: your_email@gmail.com
🔐 App Password: xxxx xxxx xxxx xxxx
📝 Message: Your message to send

That's it! Emails will start sending automatically!

================================================================================
                        🔧 INTEGRATION WITH SCRAPER
================================================================================

DATA FLOW:
──────────

Google Maps Scraper
        ↓
output/scraped_data.csv
        ↓
Email Sender
        ↓
Gmail SMTP Server
        ↓
Recipients

HOW THEY WORK TOGETHER:
───────────────────────

1. Scraper finds emails from Google Maps
2. Scraper saves emails to output/scraped_data.csv
3. Email Sender reads emails from CSV
4. Email Sender sends emails one by one
5. Tracking prevents duplicate sends
6. Daily limit enforced (500/day)
7. Automatic reset at midnight

PARALLEL EXECUTION:
───────────────────

Thread 1 (Scraper):
├─ Scrapes Google Maps
├─ Finds emails
└─ Saves to CSV

Thread 2 (Email Sender):
├─ Reads from CSV
├─ Sends emails
└─ Tracks progress

Both run simultaneously!
