                        ⚡ QUICK START (3 STEPS)
================================================================================

STEP 1: Setup Gmail (One-time only)
───────────────────────────────────

1. Go to: https://myaccount.google.com/security
2. Enable "2-Step Verification"
3. Go to: https://myaccount.google.com/apppasswords
4. Generate App Password (16 characters)
5. Copy the password

⚠️ IMPORTANT: Use App Password, NOT regular password!

STEP 2: Run Email Sender
────────────────────────

Choose ONE of these commands:

Option A (Recommended for beginners):
$ python quick_email_sender.py
└─ Shows menu with options

Option B (Direct):
$ python email_sender.py
└─ Starts email sender directly

Option C (Parallel with scraper):
$ python run_scraper_and_sender.py
└─ Runs scraper and sender together

STEP 3: Enter Configuration
────────────────────────────

When prompted, enter:
📧 Gmail address: your_email@gmail.com
🔐 App Password: xxxx xxxx xxxx xxxx
📝 Message: Your message to send

That's it! Emails will start sending automatically!
